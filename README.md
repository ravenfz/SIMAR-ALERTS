# SIMAR-ALERTS
SIMAR website scraper and twitter bot that publishes water supply issues in SIMAR network

## Install
```
make all
```

## Requirements
Pre-configure Twitter keys as system environment ;)

```
export CONSUMER_KEY=""
export CONSUMER_SECRET=""
export ACCESS_TOKEN=""
export ACCESS_TOKEN_SECRET=""
``
`
## Run
```
sh run_simar.sh
```
